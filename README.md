## ‚öôÔ∏è Key Features

Unowhy Tools empowers you with a comprehensive set of administrative tools. Here‚Äôs what you can do :

- Manage the "HiSqool Manager" service (start/stop/enable/disable).
- Remove the local "ENT" account.
- Restore the "Shell" registry value to "explorer.exe".
- Uninstall "HiSqool" and delete associated services and folders.
- Remove "silent_*.vbs.lnk" files from the startup folder.
- Clean up system folders like "TO_INSTALL", "RIDF", "OEM", and "ENT".
- Rename your PC effortlessly.
- View detailed system information.
- Assign your account (including Azure AD) as a local admin.
- Unjoin your PC from Azure AD.
- Create new local user accounts.
- Manage the built-in ".\admin" local account.
- Backup and restore drivers (via PowerShell or DISM).
- Enable, disable, or repair WinRE (Windows Recovery Environment).
- Enable Windows Hello Enterprise.
- Fix BootIM (Blue Menu).
- Access quick-launch tools (CMD, TaskMgr, REGedit, GPedit) via system tray (customizable).
- Control Windows power mode from the tray.
- Remove "IgnoreAllFailures" policy from the Boot Configuration Data (BCD).
- Enable Task Manager and account lockout policies.
- Sync Wi-Fi settings with the device‚Äôs serial number.
- Control the camera and microphone directly from the tray.
- Monitor system performance and battery status (via tray).
- Manage camera privacy settings.
- Enable Windows verbose boot status.
- Uninstall and block Microsoft Edge reinstallation.
- Enable/disable Windows Defender.
- Download driver backups from the cloud.
- Flash, backup, and customize your BIOS.
- Customize the Windows boot screen (via HackBGRT).
- Remove the "RIDF" Root CA.
- Open tools like Rufus, MAS, CRU.
- Install Microsoft 365 (Office).
- Open the HiSqool Manager panel and logs.
- Unlock BIOS with the built-in Unowhy Tools BIOS Unlocker.

**New features are added regularly !**

## üõ†Ô∏è How to Install
### Administrator privileges are required

#### **Via Winget**
1. Open the Command Prompt.
2. Run: `winget install "Unowhy Tools"`.
3. Wait for the installation to complete and enjoy the tool !

#### **Manual Installation**
1. Head to the [Releases Page](https://github.com/STY1001/Unowhy-Tools/releases/latest) and download the latest "Unowhy Tools xx.xx Setup.exe".
2. Run the setup and complete the installation.
3. Once installed, launch Unowhy Tools and start exploring its features.

### ‚ö†Ô∏è Caution: The Debug/Beta version contains experimental features and might be unstable.
